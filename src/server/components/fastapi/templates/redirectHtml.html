Welcome to XyPrissJS

The next-generation Express.js framework designed for maximum performance and developer productivity. Built by Nehonix with advanced optimization, caching, and security features.

⚡ Quick Start

📦 Installation
npm install xypriss
🔥 Basic Usage 
import { createServer } from 'xypriss';

                    const app = createServer({
                    performance: { optimizationEnabled: true },
                    cache: { enabled: true }
                    });

                    app.get('/', (req, res) => {
                    res.json({ message: 'Hello XyPrissJS!' });
                    });

                    app.start(3000);
